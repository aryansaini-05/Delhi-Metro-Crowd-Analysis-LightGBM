# Delhi Metro Crowd Analysis using LightGBM

This project focuses on analysing and predicting Delhi Metro crowd patterns using machine learning techniques. The project includes data preprocessing, exploratory data analysis (EDA), feature engineering, and predictive modelling using LightGBM.

## Features
- Data Cleaning and Preprocessing
- Exploratory Data Analysis (EDA)
- Temporal Feature Engineering
- Crowd Pattern Analysis
- Analysis using charts and graphs
- LightGBM-based Prediction Model

## Tech Stack
Python, Pandas, NumPy, Matplotlib, Seaborn, Scikit-Learn, LightGBM
